# Sound Effects for KeyQuest

This folder should contain the following sound files for the typing experience.

## Required Sound Files

| File | Description | Duration |
|------|-------------|----------|
| `keypress.mp3` | Soft mechanical key click | ~50ms |
| `keypress-wrong.mp3` | Subtle error tone | ~100ms |
| `success.mp3` | Pleasant success chime | ~300ms |
| `achievement.mp3` | Celebration fanfare | ~500ms |
| `combo.mp3` | Combo milestone sound | ~200ms |
| `level-up.mp3` | Rank/level up fanfare | ~600ms |

## Download Instructions

1. Visit one of the sources above
2. Search for appropriate sounds
3. Download as MP3 format
4. Rename to match the filenames above
5. Place in this folder (`public/sounds/`)

## Sound Guidelines

- **Keypress sounds**: Should be short, satisfying, not too loud
- **Error sounds**: Gentle, not harsh or alarming (we don't punish mistakes)
- **Success sounds**: Pleasant, celebratory, motivating
- **Achievement sounds**: More elaborate, exciting, memorable

## File Format

- Format: MP3 (or WAV for higher quality)
- Sample rate: 44.1kHz
- Bit depth: 16-bit
- Keep files small (<100KB each) for fast loading
